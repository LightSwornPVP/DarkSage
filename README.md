# DarkSage

DarkSage is a cross-platform market intelligence, trading research, portfolio, backtesting, and automated trading platform.

## Goals

- Scan and rank large stock universes
- Generate transparent trading signals
- Backtest strategies
- Track strategy performance by stock, sector, timeframe, regime, and instrument
- Build and monitor portfolios
- Support desktop and mobile clients
- Use local AI by default to minimize cost
- Begin with paper trading only
- Support carefully controlled live trading later

## Core Stack

- Desktop: Electron + React + TypeScript
- Mobile: React Native
- Backend: Python + FastAPI
- Database: SQLite initially
- Charts:
  - Apache ECharts
  - TradingView Lightweight Charts
- AI:
  - Local-first
  - Optional cloud AI
- Source Control: GitHub
- Development Orchestration:
  - Local development tools (unspecified)
