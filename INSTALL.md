# Install DarkSage Documentation Suite v1.0.0

## Target
`C:\source\repos\DarkSage`

## Recommended Safe Install
This package is additive but contains controlled files that may already exist.

1. Back up or commit the current repository first.
2. Extract this ZIP somewhere temporary.
3. Copy the **contents** of `DarkSage_Documentation_Suite_v1.0.0` into:
   `C:\source\repos\DarkSage`
4. Merge folders.
5. If Windows asks to overwrite an existing file, compare it first. Preserve newer project-specific content.
6. Confirm:
   - `docs/DOCUMENTATION_SUITE_README.md`
   - `docs/standards/`
   - `docs/templates/word/DarkSage-Document-Template.dotx`
   - `docs/templates/markdown/`
   - `docs/traceability/`

Do not nest the outer package folder inside the repository.
