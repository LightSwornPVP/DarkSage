# DarkSage Documentation Suite
**Suite Version:** 1.0.0
**Compatible Codex:** 1.x

This suite is the publishing and governance system for DarkSage documentation.

## Start Here
1. `docs/CODEX_INDEX.md`
2. `docs/standards/DOCUMENTATION_STANDARD.md`
3. `docs/standards/STYLE_GUIDE.md`
4. `docs/standards/WRITING_GUIDE.md`
5. Select a Markdown template from `docs/templates/markdown/` or a Word template from `docs/templates/word/`.

## Canonical Template Directories

- `docs/templates/markdown/` is the canonical directory for Markdown templates.
- `docs/templates/word/` is the canonical directory for Word templates.

Do not add Markdown templates directly under `docs/templates/`.

## Source of Truth
Markdown is authoritative. Word/PDF are publication editions.
